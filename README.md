# 🕯️ Attavagga: The Digital Refuge
### อัตตวรรค: ธรรมบท ขุททกนิกาย พระสุตตันตปิฎก

*"Attā hi attano nātho"* — ตนเป็นที่พึ่งแห่งตน

โครงการนี้คือการนำ **พระธรรมบท หมวดอัตตวรรค (หมวดตน)** มาถ่ายทอดผ่านผืนผ้าใบดิจิทัล (HTML/CSS) โดยเน้นความประณีตเรียบง่าย (Minimalist Artisan) เพื่อให้ผู้อ่านได้จดจ่ออยู่กับเนื้อหาในสภาวะที่จิตสงบที่สุด

---

## 🎨 Design Philosophy
- **Dark Essence:** ใช้พื้นหลังสีดำลึกเพื่อลดสิ่งรบกวนทางสายตา
- **Gold Typography:** อักษรสีทองสะท้อนความรุ่งเรืองของปัญญา
- **Flowing Presence:** ลูกเล่นค่อยๆ ปรากฏ (Fade-in) เพื่อสร้างจังหวะสมาธิขณะเลื่อนอ่าน

## 🛠️ Technology Setup
- **Pure HTML/CSS:** ปราศจากความวุ่นวายของภาษาโปรแกรมที่ซับซ้อน 
- **Responsive:** รองรับการอ่านในทุกหน้าจอ PC เพื่อความสุนทรีย์สูงสุด
